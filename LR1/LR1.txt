app является отдельным модулем для приложения и содержит все нужные файлы - код, ресурсы, картинки
Gradle Scripts служит для настроек, управления проектом 
res - используется для файлов-ресурсов разного типа: 
drawable - графические ресурсы - картинки и хмл файлы, описывающие цвет и фигуры
layout - хмл файлы, описывающие внешний вид форм и различных ее элементов
mipmap - значки приложения
values - строковые ресурсы, ресурсы цветов, тем, стилей
themes - темы 
java - содержит три папки (2 тестовые и 1 рабочая). Рабочая имеет имя проекта и содержит файлы классов
manifests - содержит файл, в котором должны быть объявлены все службы, активности, требуемые разрешения
